## Getting started

```bash
# Step 1
# Create file containing environment variables that configures app behaviour
> cp .env.example .env

# Step 2
# Install NPM dependencies
> yarn install

# Step 3
# Create CSS stylesheet
# You have to run this command whenever you edit tailwind.config.js file
> yarn run build:styles

# Step 4
# Start development server (http://localhost:3000)
> yarn run dev
```
